# _validation/ — Répertoire de validation finale

Ce répertoire centralise tous les contenus pour la validation finale
(NotebookLM + Gemini) avec un nommage explicite.

## Convention de nommage

```

```

Exemples :
- M1-script-v2.md
- M1-quiz.md
- M1-exercice.md
- M1-post-linkedin.md
