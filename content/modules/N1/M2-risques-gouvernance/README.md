# M2 — Risques & gouvernance de l'IA

> Niveau : N1 — Comprendre & situer · Statut : **À RÉDIGER**

## Description

Identifier les risques concrets liés à l'usage de l'IA générative en entreprise :
hallucinations, confidentialité, dépendance aux outils, conformité RGPD.
Poser les premières règles de gouvernance adaptées aux PME/ETI françaises.

## Compétence cible

Le manager sait identifier les 5 risques principaux de l'IA en contexte professionnel
et peut définir une règle d'usage simple pour son équipe.
