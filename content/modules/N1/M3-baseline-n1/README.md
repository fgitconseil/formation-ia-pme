# M3 — Évaluation de base Niveau 1

> Niveau : N1 — Comprendre & situer · Statut : **À RÉDIGER**

## Description

Module d'évaluation conclusif du Niveau 1.
Valider les acquis de M1 et M2 : compréhension des LLM, identification des risques,
positionnement stratégique de l'IA pour son organisation.

## Compétence cible

Le manager peut se situer par rapport à l'IA générative, nommer ses limites
et décider en connaissance de cause si — et comment — l'intégrer dans son quotidien.
