# M9 — Skills & bases de connaissance

> Niveau : N3 — Configurer & déployer · Statut : **À RÉDIGER**

## Description

Exploiter les skills et les bases de connaissance dans Claude pour enrichir
ses assistants avec des documents internes : procédures, offres, données métier.
Connecter l'IA aux ressources propres de l'organisation.

## Compétence cible

Le power user intègre des documents métier dans ses assistants IA
et crée des skills réutilisables adaptés aux workflows de son équipe.
