# M11 — Évaluation de base Niveau 3

> Niveau : N3 — Configurer & déployer · Statut : **À RÉDIGER**

## Description

Module d'évaluation conclusif du Niveau 3 et de l'ensemble de la formation.
Valider les acquis de M8, M9 et M10 : configuration d'assistants, exploitation
des skills et bases de connaissance, compréhension des MCP.

## Compétence cible

Le power user peut concevoir, déployer et maintenir un dispositif IA complet
pour son organisation, de l'assistant personnalisé aux connecteurs avancés.

Requiert M8, M9 et M10 complétés.
Module conclusif de toute la formation — prévoir une clôture forte.
