# M8 — Créer son assistant personnalisé

> Niveau : N3 — Configurer & déployer · Statut : **À RÉDIGER**

## Description

Configurer un assistant IA sur mesure adapté à son rôle et à son organisation :
définir un prompt système, un persona, des règles métier et un ton maison.
Déployer un assistant utilisable par toute une équipe.

## Compétence cible

Le power user configure et déploie un assistant IA personnalisé opérationnel
pour son cas d'usage professionnel spécifique.
