# M10 — Introduction aux MCP

> Niveau : N3 — Configurer & déployer · Statut : **À RÉDIGER**

## Description

Découvrir le protocole MCP (Model Context Protocol) et son rôle dans la connexion
de l'IA aux outils de l'entreprise : CRM, agenda, bases de données, APIs internes.
Évaluer si les MCP sont pertinents pour son organisation en V1.

## Compétence cible

Le power user comprend ce que sont les MCP, peut identifier un cas d'usage
pertinent pour son organisation et décider de les inclure ou non dans sa roadmap IA.
