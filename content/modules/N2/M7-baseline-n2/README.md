# M7 — Évaluation de base Niveau 2

> Niveau : N2 — Prompting & méthode · Statut : **À RÉDIGER**

## Description

Module d'évaluation conclusif du Niveau 2.
Valider les acquis de M4, M5 et M6 : maîtrise des types de prompts,
application de la méthode par niveaux, capitalisation des connaissances.

## Compétence cible

Le manager peut rédiger, améliorer et capitaliser des prompts professionnels
efficaces pour son activité quotidienne.
