# M4 — Les 4 types de prompts

> Niveau : N2 — Prompting & méthode · Statut : **À RÉDIGER**

## Description

Maîtriser les 4 grandes familles de prompts utilisées en contexte professionnel :
instruction, génération, analyse, transformation.
Savoir choisir le bon type selon la tâche à accomplir.

## Compétence cible

Le manager sait classer n'importe quelle tâche IA dans l'un des 4 types de prompts
et rédige un prompt adapté pour chacun d'eux.
