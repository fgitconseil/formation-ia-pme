## Variante A — Court (~500 caractères)

Il n'existe pas "un" prompt.

Il en existe 4.

Production → quand le livrable est défini.
Exploration → quand la décision n'est pas encore prise.
Structuration → quand le contenu brut existe déjà.
Vérification → quand le document doit être contrôlé.

Utiliser le mauvais type, c'est obtenir la mauvaise catégorie de réponse.
Même avec un prompt parfaitement rédigé.

C'est ça, la vraie compétence.

## Variante B — Carrousel (5 slides)

**Slide 1 — Accroche**
Pourquoi votre prompt ne donne pas ce que vous voulez.
(Ce n'est pas la formulation. C'est le type.)

**Slide 2**
Type 1 : Production.
Vous savez ce que vous voulez produire.
Email. Synthèse. Plan. Offre.
Gabarit : Rôle · Objectif · Contexte · Format · Contraintes.

**Slide 3**
Type 2 : Exploration.
Vous devez décider — mais vous n'avez pas encore les critères.
L'IA cartographie les options, hiérarchise, recommande.
Ne lui demandez pas de décider à votre place.

**Slide 4**
Type 3 : Structuration.
Vous avez du contenu brut.
Notes, transcriptions, emails désorganisés.
Règle : zéro perte d'information. L'IA organise — elle n'invente pas.

Type 4 : Vérification.
Vous avez un document. Vous voulez savoir ce qui manque.
Sans critères explicites et sans citation de sources → vérification inutile.

**Slide 5 — CTA**
4 types. 4 intentions. 4 gabarits.
Module 4 — Formation IA Générative pour PME françaises.

## Variante C — Storytelling (~1200 caractères)

Un directeur commercial m'a montré un résultat "décevant" de l'IA la semaine dernière.

Il devait comparer 3 offres fournisseurs avant un COMEX.
Il a prompté : "Analyse ces offres et dis-moi laquelle choisir."

L'IA lui a rendu une analyse structurée. Complète. Bien rédigée.
Et totalement inutile pour sa réunion.

Parce que les critères d'évaluation n'étaient pas les siens.
Parce que le contexte commercial n'était pas fourni.
Parce qu'il voulait comprendre le terrain — pas obtenir une décision automatique.

Il avait besoin d'un prompt d'exploration.
Il a utilisé un prompt de production.

Ce n'est pas une erreur de rédaction. C'est une erreur de type.

La plupart des gens pensent que la qualité d'un prompt, c'est la formulation.
C'est en fait la clarté de l'intention.

Production → créer un livrable défini.
Exploration → cartographier avant de décider.
Structuration → organiser du contenu brut.
Vérification → contrôler un document sur des critères.

Un mauvais type de prompt génère une mauvaise catégorie de réponse.
Même si chaque mot est parfait.

C'est le Module 4 de la formation que je construis pour les PME françaises.
