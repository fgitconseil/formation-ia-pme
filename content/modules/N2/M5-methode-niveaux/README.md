# M5 — La méthode par niveaux

> Niveau : N2 — Prompting & méthode · Statut : **À RÉDIGER**

## Description

Présenter la méthode de prompting structurée en niveaux de complexité croissante :
du prompt simple au prompt expert avec contexte, rôle, contraintes et format.
Progresser méthodiquement selon son niveau de maîtrise.

## Compétence cible

Le manager applique la méthode par niveaux pour améliorer itérativement
la qualité de ses prompts et obtenir des résultats fiables et reproductibles.
