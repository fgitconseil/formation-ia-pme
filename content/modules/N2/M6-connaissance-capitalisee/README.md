# M6 — Capitaliser sa connaissance

> Niveau : N2 — Prompting & méthode · Statut : **À RÉDIGER**

## Description

Construire une bibliothèque de prompts réutilisables adaptés à son métier.
Documenter les prompts efficaces, les partager en équipe, éviter de repartir
de zéro à chaque nouvelle tâche.

## Compétence cible

Le manager dispose d'une base de prompts personnalisée pour ses cas d'usage
récurrents et sait la faire évoluer au fil de son expérience terrain.

## Fichiers à créer

- [ ] `script-v1.md` — Script vidéo (~20 min, ~2 800 mots)
- [ ] `exercice.md` — Exercice pratique (construire sa première bibliothèque de prompts)
- [ ] `quiz.md` — Questions + corrigé
- [ ] `post-linkedin.md` — 3 variantes de post
- [ ] `CHANGELOG.md` — Historique des versions
